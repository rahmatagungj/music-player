# Music Player
Music player with custom playlist

## DEVELOPMENT
### Requirement
1. Node Package Manager

### Setup
```bash
npm install
```

## DEPLOY
> Make sure folder (app-rahmatagungjulians.tech) is ready

Tree :
```
  - app-rahmatagungjulians.tech
  - music-player
    - build
```


```bash
npm run build
python3 deploy.py
```
